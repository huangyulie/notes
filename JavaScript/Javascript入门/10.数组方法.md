# 1.Array.from()

用于浅拷贝一个数组

```js
let arr = [1,2,3,4,5,67];

let brr = new Array.from(arr);

console.log(arr===brr)//false;
```

# 3.数组的迭代器方法

```js
<1>keys()：返回的是数组索引

<2>values(): 返回的是数组索引的迭代器

<3>entries(): 返回的是索引值对的迭代器

这些都可以用Array.from()的方法把他们拆出来
```
